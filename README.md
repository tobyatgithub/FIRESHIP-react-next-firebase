# FIRESHIP-react-next-firebase
Use react, next, and firebase to build a blog website that supports auth, server side rendering, and history data analysis.

## Install
Make sure you are using node version 16.13.1
```
nvm install 16.13.1
nvm use 16.13.1
```

## Run
```bash
cd nextfire-app
npm run dev
```
